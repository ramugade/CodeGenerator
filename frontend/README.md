# CodeGenerator Frontend

React + TypeScript client for the CodeGenerator app.

## Getting started

Prereqs: Node.js 18+

```sh
cd frontend
npm install
npm run dev
```

The app runs on http://localhost:5173 by default.

## Environment

Set `VITE_API_URL` if your backend is not running on http://localhost:8000.

## Scripts

- `npm run dev` - Start the dev server
- `npm run build` - Production build
- `npm run preview` - Preview the production build

## Tech stack

- Vite
- TypeScript
- React
- Tailwind CSS
- Zustand
